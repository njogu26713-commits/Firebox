# FireBox Bot

FireBox Bot is a multi-purpose WhatsApp bot built with Node.js. It provides automation, AI-powered assistance, group management tools, utility commands, media features, and custom integrations to improve the WhatsApp experience.

## Features

* AI chat and assistant capabilities
* Group moderation and management
* Media download tools
* Utility commands
* Custom plugin support
* Fast and lightweight performance
* Easy deployment on cloud platforms

## Installation

1. Clone the repository:

```bash
git clone https://github.com/njogu26713-commits/firebox-bot.git
cd firebox-bot
```

2. Install dependencies:

```bash
npm install
```

3. Start the bot:

```bash
npm start
```

## Configuration

Create a `.env` file and add the required environment variables:

```env
GEMINI_API_KEY=your_api_key
OWNER_NUMBER=your_number
PREFIX=.
```

## Deployment

FireBox Bot can be deployed on:

* Railway
* Render
* VPS Servers
* Docker
* Other Node.js hosting platforms
