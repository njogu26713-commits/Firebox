Config Vars

Set your Session ID and bot settings below. Changes apply immediately.

⚡ Deploy via Session ID

  1. Get your Session ID from WhatsApp — the bot messages it to you when it first connects
  2. Paste it in the SESSION_ID field below
  3. Fill in your number and click Save & Apply — bot goes live instantly ✅
🔑 Session
SESSION_ID
Paste any Session ID. Supports Firebox, CYPHER-X, CYPHER-MD, Baileys JSON, ZIP format.
👤 Owner Settings
OWNER_NUMBER
Your WhatsApp number with country code. No + or spaces.
OWNER_NAME
Your name, shown in bot messages.
🤖 Bot Settings
PREFIX
Command prefix character. Default is .
BOT_NAME
Display name for the bot.
Saved to .env and applied live.
